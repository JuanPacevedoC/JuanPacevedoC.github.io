Gestor de Presupuestos

El proyecto desarrollado fue un gestor de presupuestos personales, creado con React y estilizado con Tailwind CSS. Su objetivo principal es permitir a los usuarios llevar un control claro y organizado de sus finanzas, identificando cuánto gastan y en qué categorías. La aplicación permite registrar gastos, clasificarlos por tipo (alimentación, transporte, entretenimiento, etc.) y visualizar un balance final que resume el estado financiero.

Contador de calorias

Este proyecto consistió en el desarrollo de una aplicación web para el conteo de calorías, construida con React y estilizada con Tailwind CSS. La herramienta permite a los usuarios registrar sus alimentos o actividades, asignarles una cantidad de calorías y organizarlos por categorías. La interfaz dinámica facilita la visualización del total de calorías consumidas o quemadas, así como el balance diario.

Calculadroa de propinas

Este proyecto consistió en el desarrollo de una aplicación web para calcular propinas de forma rápida y precisa, utilizando React para la lógica interactiva y Tailwind CSS para el diseño de la interfaz.Con cada cambio, el cálculo se actualiza automáticamente gracias al manejo eficiente del estado en React. El diseño es limpio, responsivo y fácil de usar.

Tienda de Guitarras

Este proyecto consistió en el desarrollo de una tienda online de guitarras, construida con React. La aplicación permite a los usuarios explorar una selección de guitarras, ver detalles de cada producto y agregar artículos a un carrito de compras dinámico. El manejo del estado en React fue clave para implementar funcionalidades como la actualización en tiempo real del carrito, la gestión de cantidades y el cálculo del total.

Bolg Cafe

Este proyecto consistió en el desarrollo de una página web estática para un blog de café, utilizando exclusivamente HTML y CSS. La estructura del sitio incluye secciones como "Nosotros", "Cursos" y un apartado de blog donde se comparten artículos relacionados con el mundo del café. El diseño fue trabajado cuidadosamente con CSS para lograr una interfaz atractiva, coherente y responsiva, manteniendo una buena organización del contenido y una experiencia de navegación fluida.

Proyecto-freelance

Este proyecto fue mi primera página web, desarrollada de forma independiente con HTML y CSS. Se trata de una pequeña página de presentación tipo freelance, con navegación entre secciones como inicio, servicios y contacto. Aunque no es completamente funcional, ya que algunas secciones son solo visuales, fue una base fundamental que me inspiró a seguir aprendiendo y creando proyectos más completos. A través de este ejercicio inicial, comprendí la importancia de una buena estructura HTML, la organización del contenido y el uso de estilos CSS para construir interfaces claras y agradables.
